🔥 Multiplayer Setup — takes 5 minutes, free forever
1 Create a free Firebase project
Go to console.firebase.google.com → click "Create a project" → name it anything (e.g. ipl-fantasy-2026) → disable Google Analytics → click Create project.
2 Enable Realtime Database
In your project sidebar → click Build → Realtime Database → click "Create database" → choose any region → select "Start in test mode" → click Enable.

⚠️ Set security rules — paste this in the Rules tab:
{ "rules": { ".read": true, ".write": true } }
3 Get your config keys
In Firebase console → go to Project Settings (⚙️ icon) → scroll to "Your apps" → click "Add app" → choose Web (</> icon) → register the app → copy the firebaseConfig object values below.
4 Paste your Firebase config
✅ What Firebase gives you:
Real-time auction bidding across all phones · Each friend joins on their own device · Live chat · Trade proposals with push-style updates · All data persists — no one needs to stay "host"
